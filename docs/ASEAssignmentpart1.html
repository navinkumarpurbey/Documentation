AppCanvas

Represents the application canvas for drawing shapes, lines, and text. Implements the BOOSE.ICanvas interface.

Array

The Array class inherits from the BOOSE.Array class and implements custom restrictions on the number of instances. It manages a counter for the instances created and throws an exception when the instance limit is exceeded.

CommandParser

The CommandParser class processes and interprets various commands including variable declarations, assignments, array operations, and drawing commands.

Form1

The main form of the application that interacts with the canvas and handles user inputs and program execution.

MyWhile is a subclass of the CompoundCommand class that adds restrictions on the number of instances of the class. It enforces a limit of 1000 instances.
